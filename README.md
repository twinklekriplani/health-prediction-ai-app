# Health Prediction AI Application

A comprehensive health assessment and prediction application that leverages Python and machine learning logic to analyze user metrics and deliver real-time health insights.

## 🚀 Tech Stack
- **Backend:** Python (with SQLite database management)
- **Frontend:** Responsive User Interface (HTML, CSS, JavaScript)

## 💡 Key Features (CRUD Functionality)
This application fully implements complete data management operations:
- **Create:** Securely log new user health parameters and lifestyle metrics through the clinical assessment form.
- **Read:** Display interactive AI-powered health prediction risk assessments and historical patient records.
- **Update:** Dynamically modify existing health profile metrics directly from the tracking dashboard.
- **Delete:** Permanently remove obsolete diagnostic logs cleanly from the system database.
